# ビジネスダイアグラムメーカー（Business Diagram Maker）

ブラウザだけで動作する、**単一 HTML ファイル完結**のビジネス図作成ツールです。
スイムレーン図メーカーを起点に、ノード配置・接続線ルーティング・保存/復元・各種エクスポートを
備えた汎用ダイアグラムエディタへ発展させました。インストール不要で、HTML を開くだけで使えます。

- **最新版**: [`diagram_v7.11.html`](diagram_v7.11.html)（2026-06-22）
- 開発計画は [`開発ロードマップ.txt`](開発ロードマップ.txt)、変更履歴は [`開発記録.txt`](開発記録.txt) を参照してください。

---

## 特徴

- **単一 HTML ファイル完結** — 依存サーバー不要。ブラウザ（Chrome など）で開くだけで動作します。
- **コアエンジン共通化** — ノード配置・接続線ルーティング・保存/復元・エクスポートを全図種で共用し、
  図種ごとにパレット・レーン有無・描画だけを切り替える設計です。
- **後方互換の維持** — 既存のスイムレーン図機能はすべてのバージョンで動作するよう配慮しています。

## 対応図種（9 種）

| 図種 | 補足 |
|---|---|
| スイムレーン図 | 起点となった図種。横/縦レーン、グループ枠シェイプに対応 |
| フローチャート | BPMN 記法のシェイプ・グループ枠にも対応 |
| 状態遷移図 | |
| ER 図 | |
| 樹状図 | 旧「組織図」を汎用ツリー図へ発展（v7 系） |
| シーケンス図 | |
| ガントチャート | 汎用カラムモデルにより各種工程表に対応。Excel(.xlsx) 出力対応 |
| マインドマップ | |
| UML クラス図 | |

## 主な機能

- 図形の配置・接続、接続線の直交自動ルーティング（A* 経路探索）
- 判断（ひし形）シェイプの頂点固定接続、差戻し戻り線の整形
- グループ枠シェイプ（スイムレーン／フローチャート／BPMN）
- 保存 / 復元（JSON）
- エクスポート: **PNG / SVG / PDF（印刷）/ Excel(.xlsx)**
- 検索パネル、整列・自動レイアウト、色変更、リサイズ など

## 使い方

最新版の [`diagram_v7.11.html`](diagram_v7.11.html) をブラウザで開くだけです。
過去版（`diagram_v6.16.html` 〜）も開発履歴として同梱しています。

## 開発の経緯（バージョン構成）

| 区切り | 内容 |
|---|---|
| Tier 1（v2） | コアエンジン共通化＋レーン不要図のサポート |
| Tier 2（v3） | ER 図 / 組織図 / シーケンス図 |
| Tier 3（v4） | ガントチャート / マインドマップ / UML クラス図、競合分析に基づく機能拡張 |
| v5 | 工程表対応の汎用化（汎用カラムモデル） |
| v6 | ガント/工程表の Excel(.xlsx) 出力（ExcelJS 同梱） |
| v7 | 樹状図計画（組織図 → 汎用ツリー図）、グループ枠、判断シェイプ頂点接続 ほか |

詳細な変更履歴はメジャーバージョンごとに分割しています（`開発記録_v1.txt` 〜 `開発記録_v7.txt`）。

```

開発ロードマップ*.txt   開発計画（開発ロードマップ.txt = 全体計画＋索引）

create-manual.js       操作マニュアル生成スクリプト

```

## 開発補助スクリプト（任意）

スクリーンショット取得などの補助スクリプトを使う場合のみ、依存をインストールします。

```bash
npm install        # Puppeteer を取得
node serve.js      # ローカルでプレビュー
```

本体（`diagram_*.html`）の利用自体には何のインストールも不要です。
