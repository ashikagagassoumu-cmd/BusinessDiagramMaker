# ビジネスダイアグラムメーカー — プロジェクト指示

ブラウザだけで動く、単一HTMLファイル完結のビジネス図作成ツール。最新版を直接更新していく運用。

## 開発継続の前に確認するファイル（全文を毎回読む必要はない）

新セッションで開発を継続する際は、まず以下の「索引＋現行ステータス」を確認すればよい。
詳細が必要なときだけ該当ファイルを開く。

- `開発記録.txt`（索引）→ 最新の変更履歴は `開発記録_v7.txt`（アクティブ）
- `開発ロードマップ.txt`（全体計画＋現行ステータス＋索引）
- `運用ルール.txt`（バージョニング・作業ルール）

## 現行ステータス

- 最新版: `diagram_v7.11.html`（2026-06-22）
- 次の小変更: v7.12（大変更なら v8.00）

## 運用

- 版アップ・修正後は `git add` → `git commit` → `git push`
  （GitHub private: `ashikagagassoumu-cmd/BusinessDiagramMaker`）
- **版を上げたら、この「現行ステータス」欄・`開発記録*.txt`・`開発ロードマップ.txt` を更新する**
- 微修正（1行程度）は版番を据え置き、最新版ファイルを直接更新して「追補」として記録する
  （新しいファイルを作らない）
- 設計方針: 単一HTMLファイル完結／コアエンジン共通化／既存図種の後方互換を維持
